### 慕课网技术要点
- TLS 指纹检测：requests 被拦，必须用 curl_cffi impersonate="chrome"
- 两个域名完全不同的体系：
  - www.imooc.com：`/course/playlist/{course_id}?t=m3u8&_id={mid}&cdn=aliyun1`
  - coding.imooc.com：`/lesson/m3u8h5?mid={mid}&cid={course_id}&ssl=1&cdn=aliyun1`
- coding 域还有新页面结构（`div.catague-item`），media_id 不在 HTML 里，需 API 获取
- m3u8 是 Destm 加密：XOR key_table 解密 + AES 解密，key_table 用 pop() 不是索引取值
- Windows 编码痛点：subprocess 和 stdout 都要显式设 UTF-8
